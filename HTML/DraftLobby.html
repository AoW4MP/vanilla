Draft factions, society traits, and ruler types with your friends before a match. Create a lobby to host, or open a share link to join an existing one. The default 1v1 Tournament preset randomizes Player A/B at draft start and uses blind phases for faction bans/picks and trait protections.

Advanced — Relay URL
Messages flow through a tiny WebSocket relay (AWS API Gateway WebSocket + Lambda + DynamoDB; see /aws/draft-relay.yaml). The default URL points at the production deployment. Both sides of a draft must use the same relay URL.
Changes save automatically and update the share link below. The default URL is omitted from the link; any custom URL is appended as &relay=<wss://...> so your opponent connects to the same relay.
